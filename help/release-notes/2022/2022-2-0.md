# Cloud Manager 2022.2.0版發行說明 {#release-notes}

本頁會記錄[!UICONTROL Cloud Manager] 2022.2.0版的一般發行說明。

>[!NOTE]
>
>如需AEM as a Cloud Service中Cloud Manager的最新發行說明，請參閱AEM as a Cloud Service目前發行說明中的[Cloud Manager](https://experienceleague.adobe.com/docs/experience-manager-cloud-service/content/implementing/using-cloud-manager/release-notes-cloud-manager/release-notes-cm-current.html)。

## 發行日期 {#release-date}

[!UICONTROL Cloud Manager] 2022.2.0的發行日期為2022年2月10日。 下一個版本計畫於 2022 年 3 月 10 日發行。

## 新增功能 {#whats-new}

* 可使用開發人員角色存取 AEM 環境紀錄。
* 您現在可停用 reliability_rating 關鍵量度。

## 錯誤修正 {#bug-fixes}

* 復原執行現在會和完整堆疊管道執行正確區分。
* 舊的 Git 存放庫密碼現在都一定會在新密碼產生時失效。
