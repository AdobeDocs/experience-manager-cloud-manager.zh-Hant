# Cloud Manager 2019.5.0發行說明 {#release-notes-for}

[!UICONTROL Cloud Manager] 2019.5.0版本不含重大功能變更。 如需詳細資訊，請遵循以下區段。

## 發行日期 {#release-date}

[!UICONTROL Cloud Manager] 2019.5.0的發行日期為2019年5月23日。


## 錯誤修正 {#bug-fixes}

* SLA報表圖表會在Y軸中顯示重複的值。
* 部分客戶看不到「報表」畫面。
* 在極少數情況下，無法剖析計畫碼品質結果，導致管道執行失敗。
* 管道排程步驟中的行事曆未正確本地化。
* 一系列微不足道的本地化問題和翻譯改善。
