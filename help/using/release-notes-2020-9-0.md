# 2020.9.0 版發行說明 {#release-notes-for}

以下章節概述[!UICONTROL Cloud Manager] 2020.9.0版的一般發行說明。

## 發行日期 {#release-date}

[!UICONTROL Cloud Manager] 2020.9.0版的發行日期為2020年9月3日。

## 新功能 {#whats-new}

* 建置程式已分為三個不同的Maven命令。
* 如果Git存放庫複製失敗，最多會重新嘗試三次。
* 效能測試期間使用驗證時，管道執行詳細資料頁面上現在會有指標。
