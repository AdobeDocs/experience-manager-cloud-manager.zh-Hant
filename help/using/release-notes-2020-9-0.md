# 2020.9.0 版發行說明 {#release-notes-for}

以下章節概述[!UICONTROL Cloud Manager] 2020.9.0版的一般發行說明。

## 發行日期 {#release-date}

[!UICONTROL Cloud Manager]版本2020.9.0的發行日期為2020年9月03日。

## 新功能 {#whats-new}

* 建立程式已分為三個不同的Maven命令。
* 如果Git儲存庫無法克隆，則最多將重新嘗試三次。
* 在效能測試期間使用驗證時，現在流水線執行詳細資訊頁面上有一個指示器。
